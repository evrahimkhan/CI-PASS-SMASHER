# Project Context for CI Pass Cracker

## Project Overview
The CI Pass Cracker project is a comprehensive solution for password recovery using John the Ripper within GitHub Actions. It provides automated password cracking capabilities for various file types with real-time notifications and monitoring.

## Project Goals
- Automate password recovery workflows using John the Ripper
- Support multiple file types (Office documents, PDFs, archives, etc.)
- Provide real-time notifications via Telegram
- Offer a modern UI dashboard for monitoring
- Ensure ethical usage and compliance with security policies
